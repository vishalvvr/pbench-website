### Project Setup
```
$ git clone --recurse-submodules git@github.com:vishalvvr/pbench-website.git
or
$ git clone --recurse-submodules https://github.com/vishalvvr/pbench-website.git

$ hugo server    //run local instance
$ hugo --minify  //to build static site into public dir
and commit the changes 
```
